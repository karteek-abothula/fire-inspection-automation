# fire-inspection-automation

A Python-based automation project that extracts structured fire inspection data from PDF reports and exports the cleaned results into Excel for reporting and analysis.

## Project Overview
This project was built to reduce manual effort in reviewing fire inspection PDF reports. The script reads inspection files, extracts important violation-related information, organizes the data into a structured format, and generates an Excel output that can be used for further analysis and dashboard reporting.

## Features
- Extracts fire inspection information from PDF reports
- Identifies violation-related details from unstructured text
- Organizes extracted data into a structured dataset
- Exports final results to Excel
- Supports reporting and dashboard workflows

## Tools and Libraries
- Python
- pandas
- pdfplumber
- openpyxl

## Workflow
1. Read fire inspection PDF files
2. Extract relevant inspection and violation details
3. Clean and structure the data
4. Export the final dataset to Excel
5. Use the output for analysis and dashboard creation

## Output
The final output is a structured Excel file that can be used in reporting tools such as Power BI for dashboard development and insight generation.

## Future Scope
- Automate scheduled script execution
- Enable cloud or server-based hosting
- Support automatic dashboard refresh workflows
- Improve handling of different PDF formats
